> Detect if your code is running under [PM2](https://www.npmjs.com/package/pm2).

## Install

```
$ npm install --save is-pm2
```

## Usage

```js
const isPm2 = require('is-pm2');

isPm2();
//=> 'true' if your code is running under pm2
```
